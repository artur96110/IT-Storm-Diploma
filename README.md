# IT-Storm Diploma

Дипломный проект веб-студии, разработанный на Angular.

## Функционал

* Авторизация и регистрация пользователей
* Просмотр статей блога
* Фильтрация и пагинация статей
* Просмотр детальной страницы статьи
* Добавление комментариев для авторизованных пользователей
* Лайки, дизлайки и жалобы на комментарии
* Формы обратной связи и заказа услуг
* Интеграция с backend API

## Технологии

* Angular
* TypeScript
* RxJS
* Angular Material
* SCSS
* Node.js
* MongoDB

## Запуск проекта

### Frontend

```bash
npm install
ng serve
```

### Backend

```bash
npm install
npm start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:4200
```
